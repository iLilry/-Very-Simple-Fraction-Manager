# -Very-Simple-Fraction-Manager
Im new to C and this is my first project :)
Basically, 
The manager will give u like 9 options
( 0 for exit
  1 for fraction input
  2 for fraction ouput
  3 for fraction simplification
  4 for fraction addition
  5 for fraction subtraction
  6 for fraction multiplication
  7 for fraction division
and 8 for fraction array )
I also added a special checker to force users to type a number instead of texts or symbols by using scanf and getchar() 
If users type texts or symbols it will catch the error ,reset and force them to type a valid number, no infinite loops here (i think 😂) :3

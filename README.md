# -Very-Simple-Fraction-Manager
Im new to C and this is my first project :) <br>
Basically, <br>
The manager will give u like 9 options <br>
( 0 for exit <br>
  1 for fraction input    <br>
  2 for fraction ouput    <br>
  3 for fraction simplification <br> 
  4 for fraction addition  <br>
  5 for fraction subtraction  <br>
  6 for fraction multiplication  <br>
  7 for fraction division  <br>
and 8 for fraction array )  <br>
I also added a special checker to force users to type a number instead of texts or symbols by using scanf and getchar() <br>
If users type texts or symbols it will reset and force them to type a valid number, no infinite loops here (i think 😂) :3
